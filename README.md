# pdf-to-text

## Setup and run

### 1- install dependencies

* ```
  npm install
  ```

### 2- add your api key and ocr langauge ('ara,eng,...') at index.ts

* ```typescript
  // Arabic=ara
  const ocr_language = 'ara';
  const api_key = 'add your api key here!';
  ```

### 3- add pdf files to ./files/input

### 4- run the script!

* ```
  npm run run
  ```
